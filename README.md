# touch-vue

> Touch events plugin for Vue.js(2.0)

[https://github.com/answershuto/vue-touch](https://github.com/answershuto/vue-touch)

## Installing

> npm i touch-vue

## API Usage

```javascript
import touchVue from 'touch-vue';
Vue.use(touchVue);
```

```html
<div v-touch="handleTouch">touch me</div>
```

```javascript
handleTouch (type) {
    //...
}
```
